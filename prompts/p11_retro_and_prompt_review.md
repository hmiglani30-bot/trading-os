# P11 — Retrospective and Prompt Review

## Purpose

Review a completed research cycle and identify what the workflow should learn. This prompt updates the failure log and prompt changelog.

## Required inputs

- Original report or memo.
- Outcome summary.
- Timeline of what happened.
- User notes about what felt surprising or unclear.
